# **CRUD-REACTNATIVE**

CRUD(Create, Read, Update e Delete) de um aplicativo básico de compras de supermercado, onde você informa o que vai comprar e quantidade.

Bibliotecas e IDE utilizado na aplicação:
- Visual Code
- React Native
- Json
- NodeJS

Instale o expo-cli de forma global
`$ npm install -g expo-cli`
